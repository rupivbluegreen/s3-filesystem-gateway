# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

## Our Pledge

We pledge to make participation in our project a welcoming experience for everyone, regardless of background or identity.

## Standards

We expect community members to:

- Be respectful and considerate in language and actions
- Welcome newcomers and help them get started
- Give and gracefully accept constructive feedback
- Focus on what is best for the project and the community
- Show empathy toward other community members

Unacceptable behavior includes personal attacks, harassment, publishing others' private information without permission, and any conduct that would reasonably be considered inappropriate in a professional setting.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers by opening a [private security advisory](https://github.com/rupivbluegreen/s3-filesystem-gateway/security/advisories/new) (the private reporting channel is suitable for conduct issues too) or via email to the address listed in `SECURITY.md`.

All complaints will be reviewed and investigated promptly and fairly. Maintainers will respect the privacy of reporters.

## Scope

This Code of Conduct applies within all project spaces — the repository, issues, pull requests, discussions, and any other project communication channel — and also when an individual is officially representing the project in public spaces.
